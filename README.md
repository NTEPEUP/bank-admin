## Environment variables

The app reads the API base URL from `VITE_API_BASE_URL`. If it is not defined, it falls back to `http://localhost:8080/api`.

Example local override:

```env
VITE_API_BASE_URL=http://localhost:8080/api
```

For another environment, point the same variable to the corresponding API host before running the app.
